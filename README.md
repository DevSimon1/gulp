**1. Установка пакетов:**
> npm i

**2. Запуск сборщика:**
> gulp watch

---
**Содержимое gulpfile.js**
- scssPath содержит путь к main.scss - где подключаем ВСЕ SCSS стили через import.
- scssAllFilesPath содержит все пути к SCSS файлам, для слежки(watch) за ними.
