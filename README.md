**Как пользоваться**

**1. Установка пакетов:**
> npm i

**2. Запуск сборщика:**
> gulp watch

---
**Содержимое**
- сборка умеет собирать и минифицировать SCSS, также есть автопрефиксер.
- поддерживает шаблоны html, позволяя собирать их из разных частей.
- работа с исходниками происходит в папке app.
